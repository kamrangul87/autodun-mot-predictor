Autodun MOT Predictor – Model Details

← Back to MOT Predictor

Overview

This tool estimates the probability that a vehicle will pass its next MOT test. It is designed as an early warning and decision-support tool, not as a replacement for a professional inspection.

Current Model (v1)

  • Model type: Logistic regression
  • Features used:
    • Vehicle age (years since first registration)
    • Mileage (current odometer reading)
  • Training data: DVSA-style MOT samples (pass/fail outcomes)
  • Deployment: JSON model on GitHub → API on Vercel

How the Probability Is Calculated

The model computes a score z = w_age * vehicle_age + w_mileage * mileage + intercept, then converts this into a probability using the logistic function: 

probability = 1 / (1 + exp(-z))

The current parameters are stored in ml/models/mot_model_v1.json. For example: 

{
  "coef": [[-0.25, -0.00001]],
  "intercept": [3.0],
  "columns": ["vehicle_age", "mileage"]
}

Risk Bands

  • High risk: probability < 50%
  • Medium risk: 50% – 70%
  • Low risk: probability ≥ 70%

These thresholds can be adjusted as more data is added or different calibration targets are required.

Limitations & Disclaimer

  • The current model only uses age and mileage. Other factors such as previous failures, servicing history, and vehicle type are not yet included.
  • The prediction is a statistical estimate and cannot guarantee an MOT outcome for any individual vehicle.
  • This tool is for information and experimentation purposes and should not be used as the sole basis for financial or safety decisions.
